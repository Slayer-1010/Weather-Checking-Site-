# Weather-Checking-Site-
A beautiful, responsive weather app built with HTML, CSS, and JavaScript.   Get live weather data for any city!

## Features
- Search weather by city
- Dynamic background based on weather
- Weather icons
- Temperature toggle (Celsius/Fahrenheit)
- Responsive design for mobile & desktop
